Web Development Resources

MOSAIC Web Development Resources #

Learning #

  • Khan Academy's CSS/HTML Course: easy beginner lessons that go to intermediate HTML and CSS skills
  • Codecademy: Intermediate HTML, CSS, and Javascript lessons that take you to advanced web development skills.

HTML #

  • Embed Responsively: generate the HTML code to embed YouTube or Vine videos, instagram pictures, and twitter posts

CSS #

  • Bootstrap: CSS library to make your websites look really good
  • CSS Zen Garden: the same web page with different CSS looks completely different
  • Coolors: color scheme generator
  • Material Palette: another color scheme generator

Javascript APIs #

  • Twilio: send text messages and make phone calls with javascript
  • Mailgun: send email with javascript
  • Firebase: javascript database to store information about your users and have user logins with javascript
  • Filepicker: upload files from your users

Hosting and domain names #

  • Lean Domain Search: generate available .com domain names
  • Namecheap: check if your own custom domain name (like christian.gen.co or textbooksplease.com) is available, and buy it
  • jsbin: prototype your site quickly
  • tinyurl: make a custom URL that's easy to remember

Making Money #

  • Stripe: charge credit cards
  • Google Adsense: make money on your site with ads
  • Cart.js: virtual shopping cart
  • Amazon Affiliates: make money when someone follows a link to an Amazon product and buys it (~3-10%).
  • Nike Affiliate Program: make up to 11% commission by linking to Nike products

Student Websites #

  • Why Bother Trying: Amanda's Clothing Design retail website
  • First Class Game: directory of web games by Paul
  • Click Faster Now
  • Click It: game to try to click a moving button
  • Try It: guessing game
  • Button Nyan Cat Roller: guessing game
